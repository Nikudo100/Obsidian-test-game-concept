# Гайд переноса Ravenswatch механик

1) Список героев → файлы `Герой-*` в [[05-Контент]]
2) Способности → `Способность-*` с типами урона [[03-Бой/Типы-урона]]
3) Предметы → `Предмет-*` и их синергии [[_Шаблоны/Шаблон-Билд]]
4) Враги/боссы → `Враг-*`/`Босс-*`
5) События карты → `Событие-*` [[02-Системы/События-карты]]
6) Порча → параметры в [[02-Системы/Сложность-Порча]]

Проверяй связи в [[Граф-AAA-Пример]].
