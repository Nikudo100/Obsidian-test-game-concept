# Формулы

## DPS
`DPS = (BaseDamage * (1 + PowerScaling) * (1 + CritChance*CritMult) * (1 + Modifiers)) / AttackInterval`

## DoT
`DoT_Total = TickDamage * (1 + DoTScaling) * Duration * (1 + Stacks)`

## EHP
`EHP = Health / (1 - Mitigation)` где `Mitigation = 1 - 1/(1+Armor/ArmorK)`

## Вражеский скейлинг
`EnemyPower(stage) = Base * (Scaling ^ stage)`

## Шанс дропа
`p = base * rarity_mult * pity_mult`

## Синергия билда
`S = Σ w_i * effect_i` по матрице взаимодействий [[Граф-AAA-Пример]]
